# Как добавить новую сказку

## Быстрый обзор

Сказка — это набор из 4 компонентов:

| Компонент | Формат | Зависит от языка? | Обязательно? |
|-----------|--------|:-----------------:|:------------:|
| Текст | JSON | Да | Да |
| Обложка | JPG / PNG / WebP | Нет | Да |
| Иллюстрации | JPG / PNG / WebP | Нет | Да |
| Озвучка диктора | MP3 | Да | Нет |

Одна сказка может существовать на нескольких языках с **одним и тем же `id`**.
Обложка и иллюстрации — общие для всех языков. Текст и озвучка — свои для каждого языка.

---

## Шаг 1. Написать текст сказки

Создать JSON-файл в папке нужного языка:

```
data/tales/{lang}/{id}.json
```

Пример — `data/tales/ru/repka.json`:

```json
{
  "id": "repka",
  "title": "Репка",
  "lang": "ru",
  "pages": [
    "Посадил дед репку. {childName} {m:помогал|f:помогала} поливать.",
    "Тянет-потянет — вытянуть не может. Позвал дед бабку.",
    "Тянут-потянут — вытянули репку! {childName} {m:был рад|f:была рада}!"
  ]
}
```

### Правила

- **`id`** — латиницей, через дефис (`repka`, `three-bears`). Одинаковый во всех языковых версиях.
- **`title`** — название на языке версии.
- **`lang`** — код языка, совпадает с папкой (`ru`, `en`, `kz`).
- **`pages`** — массив строк. Каждый элемент = одна страница = одна иллюстрация = один аудиофайл.

### Шаблоны персонализации

| Шаблон | Что подставляется | Пример |
|--------|-------------------|--------|
| `{childName}` | Имя ребёнка | `{childName} побежал` → `Артём побежал` |
| `{ChildName}` | Имя с большой буквы (для начала предложения) | `{ChildName} сказал` → `Артём сказал` |
| `{m:текст_м\|f:текст_ж}` | Текст по полу ребёнка | `{m:сказал\|f:сказала}` → `сказала` |

Можно использовать пустую ветку для добавления окончания: `наблюдал{m:\|f:а}` → `наблюдала`.

### Несколько языков

Для одной и той же сказки на разных языках создайте несколько файлов с **одинаковым `id`**:

```
data/tales/ru/repka.json   → { "id": "repka", "lang": "ru", ... }
data/tales/kz/repka.json   → { "id": "repka", "lang": "kz", ... }
data/tales/en/repka.json   → { "id": "repka", "lang": "en", ... }
```

Количество страниц может отличаться между языками.

---

## Шаг 2. Добавить запись в индекс

Открыть `data/tales/index.json` и добавить строку для каждой языковой версии:

```json
[
  { "id": "repka", "title": "Репка",   "lang": "ru", "file": "ru/repka.json" },
  { "id": "repka", "title": "Шалқан", "lang": "kz", "file": "kz/repka.json" },
  { "id": "repka", "title": "Turnip",  "lang": "en", "file": "en/repka.json" }
]
```

---

## Шаг 3. Добавить обложку

Положить файл:

```
data/covers/repka.jpg
```

- Одна обложка на все языки
- Рекомендуемый размер: **400 x 600 px**
- Формат: JPG (предпочтительно), PNG или WebP
- Имя файла = `id` сказки

---

## Шаг 4. Добавить иллюстрации

Создать папку и положить по одной картинке на каждую страницу:

```
data/illustrations/repka/
├── page_0.jpg
├── page_1.jpg
└── page_2.jpg
```

- Нумерация с **0**
- Количество файлов = `pages.length` (максимум среди всех языковых версий)
- Рекомендуемый размер: **1080 x 1920 px** (fullscreen portrait)
- Формат: JPG (предпочтительно), PNG или WebP
- Иллюстрации общие для всех языков

---

## Шаг 5. Добавить озвучку диктора (необязательно)

Для каждого языка создать отдельную папку с MP3-файлами:

```
data/narration/default/repka/
├── ru/
│   ├── page_0.mp3
│   ├── page_1.mp3
│   └── page_2.mp3
└── kz/
    ├── page_0.mp3
    └── page_1.mp3
```

- Нумерация с **0**
- Количество файлов = `pages.length` для данного языка
- Формат: **MP3, 128 kbps, mono, 44100 Hz**
- Текст в озвучке — **без шаблонов** (нейтральная версия, без имени ребёнка)
- Не обязательно озвучивать все языки сразу — для языков без озвучки кнопка «Слушать» просто не покажется

---

## Шаг 6. Загрузить в базу данных

```bash
node db/seed.js
```

Скрипт безопасен для повторного запуска: существующие сказки обновятся, новые добавятся.

---

## Проверка

После всех шагов можно проверить через API:

```bash
# Список сказок (должна появиться новая)
curl -H "Authorization: Bearer <token>" http://localhost:3000/api/tales?lang=ru

# Текст сказки
curl -H "Authorization: Bearer <token>" http://localhost:3000/api/tales/repka?lang=ru

# Обложка
curl -H "Authorization: Bearer <token>" http://localhost:3000/api/tales/repka/cover --output cover.jpg

# Иллюстрация страницы 0
curl -H "Authorization: Bearer <token>" http://localhost:3000/api/tales/repka/illustration/0 --output page_0.jpg

# Проверить наличие озвучки
curl -H "Authorization: Bearer <token>" http://localhost:3000/api/tales/repka/default-narration?lang=ru

# Скачать озвучку страницы 0
curl -H "Authorization: Bearer <token>" http://localhost:3000/api/tales/repka/default-narration/0?lang=ru --output page_0.mp3
```

## Итоговая структура файлов

```
data/
├── tales/
│   ├── index.json                          ← добавить записи сюда
│   ├── ru/
│   │   └── repka.json                      ← текст (ru)
│   ├── kz/
│   │   └── repka.json                      ← текст (kz)
│   └── en/
│       └── repka.json                      ← текст (en)
├── covers/
│   └── repka.jpg                           ← обложка (одна на все языки)
├── illustrations/
│   └── repka/
│       ├── page_0.jpg                      ← иллюстрации (общие)
│       ├── page_1.jpg
│       └── page_2.jpg
└── narration/
    └── default/
        └── repka/
            ├── ru/
            │   ├── page_0.mp3              ← озвучка (ru)
            │   ├── page_1.mp3
            │   └── page_2.mp3
            └── kz/
                ├── page_0.mp3              ← озвучка (kz)
                └── page_1.mp3
```
